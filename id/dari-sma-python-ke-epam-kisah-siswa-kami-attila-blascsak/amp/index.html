Dari SMA Python ke EPAM – Kisah siswa kami, Attila Blascsák.

Attila Blascsák adalah salah satu pengguna pertama CodeBerry, yang berhasil mengubah karirnya dan menjadi pengembang web setelah belajar bersama kami. Kami mengobrol dengan Attila tentang bagaimana dia sampai di sini, apa yang dia pelajari dan apa kesulitan mencari pekerjaan sebagai pengembang junior.

Ketika seseorang bertanya tentang profesimu beberapa bulan yang lalu, bagaimana kamu menjawabnya?

Saya biasa mengatakan pekerjaan saya adalah melakukan networking di Layanan IT, yang berarti saya membantu perusahaan besar bergabung dengan internet. Tugas kami bervariasi dari memastikan router sampai ke pelanggan hingga membuatnya benar-benar bekerja.

Dan apa jawabanmu sekarang?

Saya adalah pengembang UI front-end di EPAM. Saya bekerja di tim yang lebih besar, mengembangkan aplikasi yang utamanya mengandalkan pada teknologi front-end dan merancang antarmuka pengguna (user interface). Saya menggunakan HTML, CSS dan JavaScript setiap hari.

Kedengarannya hebat! Jadi, pemrograman bukanlah tugas baru bagimu, kan? Apakah kamu pernah melakukan coding sebelumnya?

Ya, saya mulai menggunakan HTML dan CSS ketika SD. Kemudian di SMP, kami selalu menggunakan Python, mengandalkan sebuah buku antik. Saya pun berhenti melakukannya karena perpustakaan yang dikutip oleh penulis tersebut sudah tidak ada lagi, sehingga tugasnya tidak dapat diselesaikan. 🙂

Dari mana ide pengembangan web berasal?

Selama saya di sekolah menengah atas, dua orang mendekati saya dan mengatakan mereka membutuhkan pengembang web untuk startup mereka. Saat itu, saya tidak tahu apa yang akan saya dalami, tetapi saya menjawab ya dan saya dapat segera menguasainya. Akhirnya kami menyelesaikan proyek tersebut, tetapi pengembangan web menarik perhatian saya, jadi saya terus mempelajarinya sendiri. Akhirnya, saya mengalami jalan buntu di musim semi, ketika saya merasa tidak bisa melakukannya lagi, saya butuh seorang guru. Saat itulah saya menemukan CodeBerry. 🙂

Sudah berapa lama kamu belajar dengan CodeBerry?

Sejak Juni 2016.

Dan mengapa kamu menyukainya, apa kelebihannya?

Sejujurnya, tidak ada jumlah informasi baru yang luar biasa, tetapi tentu saja sangat bagus untuk mengingat kembali pengetahuan saya. Para mentor banyak membantu saya – itu mungkin salah satu hal terbaik di CodeBerry. Memiliki seseorang untuk mendiskusikan masalahmu adalah sebuah keuntungan besar.

Senang mendengarnya. Sekarang awal musim gugur, itu berarti kamu telah memperoleh cukup pengetahuan dalam 3 bulan untuk mencari pekerjaan. Berapa lama waktu yang kamu butuhkan untuk menemukannya? (Wawancara berlangsung pada September 2016 – ed.)

Sebenarnya, saya bahkan belum memutuskan untuk mulai mencari pekerjaan. 🙂 Saya membuat profil LinkedIn, di mana saya menyatakan bahwa saya sedang belajar untuk menjadi pengembang junior dan menambah keterampilan baru saya juga. Dua minggu kemudian, EPAM menghubungi saya. Saya baru akan membuat portofolio sebelum fokus mencari pekerjaan, tetapi EPAM lebih cepat. Mereka langsung mengundang saya untuk wawancara.

Saya ingin bertanya apakah pencarian itu sulit, tetapi saya kira kita bisa melewati pertanyaan ini.

Iya. (Tertawa)

Apa langkah selanjutnya untukmu? Arah apa yang kamu ambil?

React environment adalah favorit saya saat ini, saya ingin belajar lebih banyak tentangnya di periode mendatang. Ini juga mimpi besar saya untuk mengajarkan pemrograman dalam beberapa bentuk. Memberikan kuliah atau mengambil bagian dalam program yang lebih spesifik, saya belum tahu, juga tidak tahu bagaimana itu akan terjadi, tetapi saya yakin itu akan terwujud suatu saat nanti.

Sebagai siswa yang berpengalaman, apakah kamu memiliki kiat untuk memulai coding?

Saya rasa penting untuk memahami bahwa proses pembelajaran tidak dapat benar-benar dipercepat. Pemrograman adalah tantangan besar, tidak mudah di awal. Tetapi setelah kamu mendalaminya dan memahami logikanya, semuanya menjadi jelas. Mulai saat itu dan seterusnya semuanya jadi dipercepat. 🙂

Dalam serial “Kisah Sukses KodeBerry”, kami berbicara dengan siswa kami yang telah mengubah karir mereka dan sekarang bekerja sebagai seorang pengembang. Jika kamu ingin tahu apa yang mereka lakukan untuk mencapainya, tetap ikuti episode selanjutnya!

Apakah kamu ingin belajar coding? Datang dan coba pelajaran pertama kami secara gratis di Sekolah Pemrograman CodeBerry.

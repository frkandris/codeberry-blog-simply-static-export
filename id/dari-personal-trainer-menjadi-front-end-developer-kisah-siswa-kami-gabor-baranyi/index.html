Dari Personal Trainer menjadi Front-End Developer – Kisah Siswa Kami, Gábor Baranyi

apa itu frontend apa itu personal trainer bahasa indonesia front belajar front end belajar front end developer belajar front end web blog frontend cara menjadi personal trainer codecademy indonesia contoh front end contoh front end dan back end contoh front end web front end developer indonesia front-end frontend adalah kursus front end developer Pelatih Pribadi Pengembang peter baranyai fan programmer sekolah personal trainer trainer kami tugas back end developer tugas front end developer

Gábor Baranyai pernah bekerja sebagai personal trainer, dan juga mengadakan sesi pelatihan untuk kelompok di gym. Sejak itu, ia telah menemukan sesuatu yang memenuhi keinginan dirinya, dan di mana ia dapat terus berkembang: ia bekerja sebagai seorang pengembang frontend junior di sebuah perusahaan menengah.

 

Saya tinggal di: Szeged, Hungary

Saya berkerja di: Minero IT Hungary KFT.

Lama belajar di Codeberry: 10 bulan

Saya bekerja menggunakan: Fujitsu Lifebook A555, Windows 7, IntelliJ, SourceTree

 

Ketika kamu ditanya tentang profesimu, bagaimana kamu menjawabnya?

Itu tergantung pada siapa yang bertanya. Jika seorang profesional di bidang ini, saya katakan saya bekerja sebagai pengembang frontend. Jika orang awam yang tertarik, jawaban saya adalah saya seorang programmer atau pengembang perangkat lunak.

 

Apa pekerjaanmu sebelum beralih menjadi pengembang web?

Saya dulu bekerja penuh waktu sebagai personal trainer, dan mengadakan sesi pelatihan kelompok di gym fungsional. Saya masih menawarkan sesi grup ini, selain bekerja sebagai programmer.

 

Mengapa kamu memutuskan untuk mulai mempelajari pengembangan web?

Saya selalu tertarik dengan desain web, bagaimana dan mengapa situs web terlihat seperti itu. Selain itu, halaman web gym sebagian dikelola oleh kami, para pelatih / trainer, jadi saya menghabiskan banyak waktu untuk menulis dan mendesain blog WordPress kami, yang sangat saya nikmati. Saya melihat peluang bagus di Facebook, di mana mereka mengiklankan kursus pengembangan web online. Saya mencari tahu, dan saya tahu bahwa saya harus menguasai seluk-beluk profesi ini di platform ini … atau setidaknya dasar-dasarnya. Itu adalah iklan dari CodeBerry.

 

Bagaimana cara kamu belajar pemrograman? Jenis kursus, buku, atau apa yang kamu gunakan?

Setelah melihat iklan yang saya sebutkan di atas, saya mulai belajar di kelas online CodeBerry pada bulan September 2016, dan kemudian saya menambahkan kursus Codecademy dan Udemy ke sumber pembelajaran saya juga. Saya mencari dan mempelajari bahasa pemrograman tertentu di situs berbagi video.

 

Apa yang ditawarkan oleh CodeBerry? Bagaimana rasanya belajar di sekolah kami?

Itu sangat berguna. Saya sangat menyukai sistem berlangganan bulanan CodeBerry, karena dengan cara ini saya bisa belajar dengan kecepatan diri saya sendiri. Saya menyukai penjelasannya, contoh-contoh melalui tugas-tugas yang disajikan.

Meja kerja Gábor

 

Apakah kamu memiliki pengalaman, tips, atau trik untuk berbagi dengan mereka yang baru mulai mempelajarinya?

Saya merasa sangat penting untuk mempelajari dasar-dasar dengan baik dan merevisi pelajaran sebelumnya untuk melihat hubungannya dan untuk mampu menempatkan materi baru dalam konteks. Revisi sangat penting.

 

Ceritakan sedikit tentang di mana kamu bekerja sekarang, apa tugasmu. Apa yang biasanya kamu lakukan tiap hari atau tiap minggu?

Saat ini, saya bekerja di kantor perusahaan yang berbasis di Budapest di daerah Szeged. Kami sedang mengembangkan proyek internal di mana saya berurusan dengan frontend, yang terdiri dari HTML, CSS, dan JavaScript.

Saya tiba di kantor sekitar jam 8 pagi dan saya pulang jam 5 sore. Saya mencoba melakukan sesuatu yang berbeda ketika saya tidak ada di sana; dengan latihan atau mengadakan pelatihan kelompok, sehingga saya dapat menghadapi hari kerja berikutnya dengan berenergi dan segar bugar.

 

Dengan siapa kamu bekerja, dan bagaimana mereka membantu pekerjaanmu?

Saat ini ada sekitar 20 orang yang bekerja di kantor. Saya memiliki seorang mentor yang membantu saya dengan perkembangan saya, yang juga mengawasi proyek yang sedang kami kerjakan.

 

Software, alat atau teknologi apa yang kamu gunakan saat ini?

Lingkungan pengembangan adalah IntelliJ, untuk kontrol versi kami menggunakan SourceTree dan untuk repo, kami menggunakan BitBucket.

Dalam proyek kami menggunakan HTML, CSS (SCSS), jQuery, Angular JS, Angular Material dan FUSE.

 

Berapa lama kamu mencari pekerjaan? Apakah sulit menemukan tempat untuk bekerja?

Sebenarnya, saya masih belajar ketika teman saya menyarankan saya untuk mengirim CV saya ke perusahaan tempat dia bekerja. Mereka mengundang saya untuk wawancara, dan beberapa minggu kemudian mereka menelepon saya mengatakan bahwa mereka ingin bekerja dengan saya, berdasarkan kinerja kerja saya.

 

Bagaimana proses wawancara?

Setelah pengenalan singkat perusahaan, wawancara dilanjutkan dalam bahasa Inggris, dan giliran saya memperkenalkan diri, menceritakan sedikit tentang kehidupan pribadi saya dan tentang bagaimana dan di mana saya belajar pemrograman, juga dalam bahasa Inggris.

Ini diikuti oleh bagian teknis, di mana mereka menilai pengetahuan saya, dan memberi saya beberapa tes logika untuk dipecahkan.

Seluruh wawancara memakan waktu sekitar satu jam.

 

Apakah kamu punya saran untuk mereka yang sedang mempersiapkan wawancara kerja?

Sangat penting bahwa semua yang kamu katakan harus positif. Bahkan jika sesuatu yang negatif muncul, cobalah membicarakannya dari segi atau perspektif positifnya. Selalu tetap percaya diri, dan tunjukkan bahwa kamu mampu dan berkeinginan untuk meningkatkannya dalam bidang apa pun.

 

Apa langkah kamu selanjutnya? Arah apa yang kamu ambil?

Saya ingin tahu sebanyak mungkin tentang pengembangan frontend, dan merasakan juga pengembangan backend untuk melihat dan memahami seluruh proses latar belakang.

 

Dalam seri “Cerita Siswa CodeBerry” kami, kami mengobrol dengan para siswa kami yang berhasil menemukan pekerjaan sebagai pengembang.

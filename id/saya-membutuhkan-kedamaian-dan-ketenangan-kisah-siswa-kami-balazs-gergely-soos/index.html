Saya membutuhkan kedamaian dan ketenangan – Kisah siswa kami, Balázs Gergely Soós

bahasa inggris kedamaian bahasa inggris ketenangan balázs gergely Budapest cerita 23++ Hungaria insinyur ilmu komputer insinyur penelitian kedamaian ketenangan ketenangan in english MorganStanley pengembang senior saya membutuhkan

Balázs Gergely Soós, BeeGee adalah lulusan insinyur ilmu komputer dengan gelar PhD dari PPKE Multidisciplinary Doctoral School of Sciences and Technology , juga anggota alumni dari BME Schönherz College. Dia sebelumnya bekerja untuk SZTAKI sebagai asisten peneliti, insinyur penelitian di Carilab, pengembang senior dan pendiri Streamnovation dan pemimpin programmer di Zinemath.

Saya tinggal di Budapest, Hungaria
Saya bekerja di MorganStanley
Waktu yang dihabiskan belajar dengan CodeBerry adalah 3 bulan
Saya bekerja dengan HP ENVY, Samsung Galaxy S4 mini, Chrome, Notepad ++, gitk

Ketika kamu ditanya tentang profesimu, bagaimana kamu menjawabnya?

Saya adalah pengembang perangkat lunak.

Apa pekerjaanmu sebelum memulai coding? Apakah kamu memiliki pengalaman sebelumnya di bidang ini?

Ketika saya masih kecil saya sering mencari kesalahan O dan nol dalam program BASIC. Saya meninggalkan bengkel pemodelan pesawat untuk menggambar dengan Commodore + 4 dan saya sering mengotak-atik GV-Basic (C64) juga. Saya menggunakan Turbo Pascal di sekolah menengah dan C++ di universitas di antara bahasa-bahasa menarik lainnya. Saya menggunakan C++ dalam waktu yang cukup lama… dan kemudian visi tentang mesin, robotika dan pemrograman GPU pun berdatangan.

Mengapa kamu memutuskan untuk mulai mempelajari pengembangan web?

Menurut saya, aplikasi desktop sudah kuno. Sebaliknya, saya tertarik dengan pengembangan seluler dan layanan cloud. Saya ingin belajar WebGL setelah menemukannya untuk pertama kalinya. Saya pikir langkah pertama harus mengingat pengetahuan web saya, karena saya sebelumnya telah berurusan dengan beberapa PHP dan MySQL, tetapi front-end selalu dilakukan oleh orang lain. Banyak proyek saya berfokus pada perhitungan, bukan interface. Namun, saya menjadi penasaran dengan trik yang dilakukan oleh para ahli GUI.

Beri tahu kami lebih banyak tentang apa yang kamu lakukan sekarang, tugas apa yang kamu miliki? Apa yang kamu kerjakan sehari-hari?

Saya bekerja di bank, jadi itu rahasia. 🙂 Tapi, pekerjaan saya meliputi beberapa tugas harian front-end.

Berapa lama kamu mencari pekerjaan? Bagaimana kamu berada pada posisimu sekarang?

Pekerjaan saya sebelumnya adalah seperti pekerjaan impian: seorang perencana pemimpi sejati, insinyur hebat, pameran internasional, dan investor yang hebat. Tapi, suatu hari saya merasa sepertinya itu terlalu banyak. Saya membutuhkan kedamaian dan ketenangan untuk dapat mendengar beberapa suara baru dan akhirnya memiliki waktu untuk keluarga saya dan saya sendiri.

Keinginan saya ini timbul saat musim panas. Istri saya libur 4 minggu, jadi kami pindah ke Venesia untuk sementara waktu. Bersantai di tepi danau membuat saya menyadari betapa saya kehilangan beberapa aktivitas intelektual, jadi saya mulai belajar…

Dua bulan kemudian, saya melakukan beberapa wawancara. Ketika berita tentang saya tidak bekerja tersebar, salah satu rekan asrama saya menelepon saya dan berkata jika saya masih menyukai bahasa program khusus tersebut dan saya tertarik dengan basis data yang besar, maka dia memiliki meja kosong untuk saya di sisinya…

Bagaimana cara kamu belajar pemrograman? Jenis kursus apa, buku apa, dll yang kamu gunakan?

Saya tidak ingat lagi, itu terjadi begitu saja. Saya memiliki beberapa masalah pada Computerbooks. Hal berikutnya adalah universitas: György Pongor dan Zoltán László. Setelah itu, saya melakukan materi online.

Apa yang ditawarkan oleh CodeBerry? Bagaimana rasanya belajar di sekolah kami?

Saya membaca tentang Node.js dan Angular dan dapat melihat kemampuan dari kombinasi keduanya. Selama era doktor saya, saya jatuh cinta pada penelitian independen, kesenangan untuk menemukan sesuatu yang baru. Di sisi lain, saya menghormati guru yang memberi saya dorongan sebelum perjalanan karir saya. Belajar dari blog benar-benar seperti berada di kota besar tanpa peta: ada puluhan bus, banyak jalur trem, dan beberapa kereta bawah tanah. Terkadang kamu perlu bantuan dengan petunjuk jika kamu benar-benar ingin pergi ke suatu tempat.”

Andris dan saya bertemu setiap bulan. Dia memberitahuku tentang CodeBerry yang ketika itu hanya merupakan sebuah binar di matanya. Belakangan Bini mengatakan dia menyukai gagasan itu. Saya pikir: mengapa tidak? Saya mungkin adalah salah satu dari dua puluh pelamar pertama. 🙂

Sebelumnya bukan tugas saya untuk membuat sesuatu yang cantik. Di awal pelajaran, fokusnya adalah pada bagian depan. Saya tidak menyesali itu. Sangat berguna untuk mencoba semua yang baru saya pelajari dan menguji pengetahuan saya dengan kuis. Pada saat saya menyelesaikan situs WordPress, CSS dan saya menjadi teman baik.

Apakah Anda memiliki pengalaman, tips, atau trik untuk berbagi dengan mereka yang baru memulai studi mereka?

Kamu hanya perlu duduk dan melakukannya… Saya mendapatkan dorongan terakhir dari seorang teman: dia meminta saya untuk membuat situs web yang responsif untuknya. Dia memanfaatkan saya yang saat itu sedang tidak bekerja. 🙂

Apa langkah selanjutnya untukmu? Arah apa yang kamu ambil?

Segera setelah saya memiliki beberapa akhir pekan kosong, saya akan kembali ke tujuan awal saya, WebGL. Mungkin dibumbui dengan beberapa VR.

Dalam seri “Cerita Siswa CodeBerry” kami, kami mengobrol dengan para siswa kami yang berhasil menemukan pekerjaan sebagai pengembang.

Apakah kamu ingin belajar coding? Datang dan coba 25 pelajaran pertama kami secara gratis di Sekolah Pemrograman CodeBerry.

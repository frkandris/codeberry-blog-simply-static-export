Mi az a Java? 

A Java (nem összekeverendő a JavaScripttel) egy általános célú programozási nyelv, amit arra terveztek, hogy sokféle eszközön fusson, az okostelefonoktól és PC-ktől kezdve egészen az okostévékig. A Stack Overflow fejlesztői felmérése alapján a Java az 5. legnépszerűbb programnyelv, a sorban közvetlenül az SQL után.

 

 

 

A Java programozás még ma is népszerű, bár már 1995-ben elkészítette a Sun Microsystems nevű cég. A Java a C++ egyszerűbb változata, ami sok tervezésbeli elemet átvett az elődjétől, az Oaktól. A Java mottója a „WORA”, ami annyit tesz: „Write Once, Run Anywhere,” azaz „Írd meg egyszer, futtasd bárhol”. Miután a kódot megírod, bármilyen kompatibilis eszközön futtathatod, amin a Java telepítve van, újrafordítás nélkül.

 

Egy jó ötlettel és némi Java programozással csodás dolgokat alkothatsz rövid idő alatt is

Így néz ki egy Java kód a klasszikus „Helló, világ!” példát használva:

class HelloWorldApp {

    public static void main(String[ ] args) {

        System.out.printIn(“Helló, világ!”); // Megjeleníti a szöveget a képernyőn. 

    }

}

 

Milyen Javában programozni? 

A Javával képesek vagyunk objektumorientáltan programozni, azaz az attribútumokat objektumokba és osztályokba rendezhetjük. Amint definiálod ezeket a programodban, később utalhatsz rájuk, így nem kell újra leírnod az egész kódot. A procedurális programozási nyelvekben mindig ki kell írni az egymástól függő hosszú kódsorokat. A praktikus ki objektumaink nélkül mindig újra le kell írni mindent.

 

A Java felhasználása

Az a jó a Java programozásban, hogy rendkívül sokoldalú, és szinte mindenhol megtalálható az iparban. Sőt, ha megtanulod a Javát, soha nem fogsz kifogyni a felhasználási és alkalmazási lehetőségekből. A Javát leggyakrabban webfejlesztésre, valamint mobil- és asztali alkalmazások fejlesztésére használják.

Mobilalkalmazások

Javával teljes Android-alkalmazásokat lehet létrehozni, így ha jól ismered ezt a nyelvet, akár Android-fejlesztő is lehetsz.

 

Webfejlesztés

A weboldalak általában HTML-ből, CSS-ből és JavaScriptből állnak, de a webes alkalmazásokat többféle programozási nyelven is meg lehet írni. Sok webes alkalmazás támaszkodik például Javában írt szerverkódokra. A Javával emellett kisalkalmazásokat (Appleteket) is lehet írni, például játékokat, amik közvetlenül a böngészőben futtathatók. Utóbbiról ezen az oldalon olvasgathattok egy kicsit bővebben, illetve a Java számos szuper összetevőjéről is megtudhattok érdekes dolgokat.

 

Asztali alkalmazások

Az „Írd meg egyszer, futtasd bárhol” filozófiát szem előtt tartva a Java asztali alkalmazások futtatására is használható, például Windowson, MacOS-en és Linuxon. A Java valójában egyik platformon sem natív nyelv, de rengeteg olyan eszköz érhető el, amelyek segítségével bármilyen operációs rendszeren futtatható. A Sun Microsystemset megvásárló Oracle nevű cég fejlesztői eszközöket is kínál Java-programok írásához különféle platformokra.

 

 

Vállalkozás

A Java nem csak programnyelv, hanem platform is. A Java EE, azaz a Java Enterprise Edition  a Java különleges verziója, amit üzleti alkalmazásokhoz terveztek. A Java EE-t gyakran használják nagy méretű hálózatokhoz, és magasabb szintű biztonságot kínál a vállalatoknak.

 

 

Szerverek

Végül, de nem utolsósorban a Javát szerverekhez is használják. A Java EE mellett az Oracle Server JRE-t is kínál, ami egy kimondottan szerverekhez tervezett Java platform. A JRE a „Java Runtime Environment” rövidítése.

 

Mennyit keresnek a Java programozók?

A Payscale alapján az egyesült államokbeli Java-programozók átlagosan évi 72 611 dollárt keresnek, ami körülbelül havi 1,8 millió forintnak felel meg. Ahogy te is tudhatod, az egyesült államokbeli programozók keresnek a legjobban.

 

 

Bár valószínűleg Magyarországon ennél lényegesen kevesebbet tudsz csak keresni ezzel a tudással, még így is hihetetlen kereseti lehetőségeid lesznek, ha megtanulsz Javában programozni. A malajziai Java-programozók például akár 1200%-kal is többet kereshetnek az országuk átlagkereseténél. Nem semmi. Hogyan tanulhatom meg a Javát?

A népszerűsége és nagy kereslete miatt sok online és offline forrás érhető el, aminek segítségével megtanulhatod a Java programozás alapjait. Ha többet szeretnél tudni erről, olvasd el a „Hogyan tanuljunk meg programozni?” című cikkünket.

 

Mennyi idő alatt lehet megtanulni a Javát? 

A programozási nyelvek megtanulásának ideje nagyban függ attól, milyen módszereket használsz, mennyi mindent tudsz megjegyezni, és hetente hány órát szánsz a tanulásra. Ne várd, hogy mindent tudni fogsz a C++-ról egy féléves egyetemi képzés vagy egy 14 hetes programozóképzés elvégzése után. A megfelelő hozzáállással azonban viszonylag gyorsan szert tehetsz egy biztos alaptudásra.

 

 

 

Honnan tudhatom, hogy nekem való-e a Java?

Az az egyik legjobb dolog a programozásban, hogy sokféle nyelv létezik. Ha utálsz például C♯-pal programozni, sok más alternatíva közül választhatsz – áttérhetsz például a Java nyelvre. A tanulásnak sosincs vége.

A legtöbb embernek jó kezdés lehet a Java, mert a segítségével sokféle eszközre programozhatsz, sokféle alkalmazást készíthetsz egyetlen nyelvvel. 

Viszont ha most kezded a programozást, és nem tudod, jó választás-e neked a Java, olvasd el a „Melyik programozási nyelvet válasszam?” című cikkünket!

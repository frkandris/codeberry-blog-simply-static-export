CSS background image című bejegyzésünkben a weboldalak hátterének elkészítéséhez gyűjtöttük össze a legfontosabb CSS beállításokat. Milyen tulajdonságok (property) adhatók meg, és milyen értékeket (value) vehetnek föl?

Az egyszínű hátterek − CSS background color

A háttérszínt a background color property segítségével állíthatjuk be. Ezt a tulajdonságot nem csak egyszínű háttereknél tudjuk alkalmazni, de akkor is hasznos lesz, mikor olyan háttérképet állítunk be, ami nem tölti ki az egész képernyőt. Ilyenkor a fennmaradó területeknek szintén ezzel az utasítással tudunk színt adni. A színek megadásának módjairól és a színkódokról ebben a bejegyzésben olvashatsz részletesen. 

A példában a háttér színét hexadecimális értékkel adtuk meg, a betűk színét pedig a sztenderd színnevek egyikével. Mivel az egész oldalra vonatkozik, ezért a body selector után kerül.

Egyes elemek (táblázat, szövegdoboz, stb.) háttérszínének beállítása

A fent leírt módon történik, csupán azzal a selectorral együtt adjuk hozzá CSS kódhoz, amelyik elemre vonatkozik. Ha fent látott táblázat háttérszínét szeretnénk megváltoztatni, a table után írjuk a background colort.

Háttérkép beállítása – CSS background image

A background image property-vel egy képet adhatunk meg a weboldal vagy egy adott elem (pl. egy div, vagy egy táblázat) háttereként. Szükséges megadni a kép elérési útvonalát, pl. az url-t  vagy mappát, ahol a kép megtalálható.

Az oldal háttérképének beállítása

Ebben a videóban láthatod, hogyan néz ki a háttérkép beállítása a gyakorlatban. Ez a másik, angol nyelvű videó pedig további példákon keresztül mutatja be a háttérformázást.

Háttérkép-méretezés – CSS background-size

A háttérkép méretére vonatkozó információkat a background-size propertyvel adhatjuk meg pixelpontban, vagy a teljes képernyőhöz viszonyított százalékarányban. 

A háttérkép méretének beállítása

Ha azt szeretnénk, hogy képünk a hátteret mindig a teljes egészében kitöltse, a pontos méretek helyett adjuk meg a  “cover” utasítást – azonban érdemes vele óvatosan bánni, mivel ez a beállítás a kép megnyúlását vagy a széleinek levágását eredményezheti.

Ha pedig biztosra akarunk menni, hogy a képünk mindig teljes egészében látszani fog a háttérben, a “contain” utasítást írjuk a méret helyére.

A háttérkép ismétlése – CSS background-repeat

Alapesetben  a kép ismétlődik, hogy kitöltse az egész oldalt, vagy az egész elemet. A lenti példában ezt láthatjuk. 

Ismétlődő háttérkép – background-repeat

CSS background-repeat: no-repeat

Ha csak egyszer szeretnénk megjeleníteni az adott háttérképet, a background repeat-nél no-repeat-et kell megadnunk. 

A háttérkép pozicionálása – CSS background-position

Alapesetben a háttérkép a bal felső sarokban jelenik meg. Ha máshová szeretnénk pozicionálni, a background-position-t kell használnunk.

Vízszintesen és függőleges is háromféle elhelyezkedést adhatunk meg: left, center, right (bal, közép, jobb) illetve top, center, bottom (fent, középen, alul).

Rövid utasítások – CSS shorthands

Rövid utasításokat használva a következő sorrendben adhatjuk meg a tulajdonságokat: 

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position

Ha valamelyik tulajdonságot kihagyjuk, az utasítások akkor is érvényesek, ha többi sorrendje megfelelő. 

Tehát pl. a következő utasításokat:

  1. background-color: #000;
  2. background-image: url(images/bg.gif);
  3. background-repeat: no-repeat;
  4. background-position: left top;

Megadhatjuk egy sorban:

background: #000 url(images/bg.gif) no-repeat left top;

Ezekkel a beállításokkal már klassz hátteret adhatsz a weboldaladnak, és egyszerűen kivitelezheted a saját elképzeléseid.
